# 📦 Sistema de Controle de Estoque

Sistema desenvolvido em Python para praticar lógica de programação, utilizando listas, dicionários, funções e tratamento de exceções.

---

## 🚀 Funcionalidades

- ✅ Cadastro de produtos
- ✅ Consulta de produtos
- ✅ Venda de produtos
- ✅ Controle de quantidade em estoque
- ✅ Validação de entradas do usuário
- ✅ Menu interativo no terminal

---

## 🛠️ Tecnologias utilizadas

- Python 3

---

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/Roniel1997/sistema-estoque-python.git
```

2. Entre na pasta do projeto:

```bash
cd sistema-estoque-python
```

3. Execute o programa:

```bash
python controle_estoque.py
```
